3.3.1 Organisation

Le tableau 3.9 présente le backlog du troisième sprint, centré sur le dépôt et le
contrôle documentaire, la validation douanière assistée par IA, et les actions
transitaire / courtier sur les expéditions.

Périmètre retenu :
  • Gérer Documents      — téléversement, consultation, téléchargement (exportateur,
                          transitaire, parties au dossier)
  • Douane (IA)          — file d'attente courtier, vérification OpenAI, validation /
                          rejet avec motif
  • Expéditions (suite)  — vue transitaire, mise à jour de statut (moteur de transitions)
  • Cockpits             — transitaire (8.3), courtier en douane (8.4)

Reporté : notifications automatiques sur documents → Sprint 4 ; messagerie → Sprint 4.
