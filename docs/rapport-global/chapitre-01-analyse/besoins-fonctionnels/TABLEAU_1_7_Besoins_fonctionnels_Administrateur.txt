================================================================================
TABLEAU 1.7 – Besoins fonctionnels de l'acteur « Administrateur »
Projet : GlobalTradeX — Plateforme de pilotage du commerce international
================================================================================

+----------------------------------+----------------------------------------------------------+
| Fonctionnalité                   | Description                                              |
+----------------------------------+----------------------------------------------------------+
| S'authentifier                   | Pour accéder à son espace, l'administrateur s'authentifie|
|                                  | avec son adresse e-mail et son mot de passe. Un jeton    |
|                                  | JWT ouvre le tableau de bord central                     |
|                                  | (/dashboard/admin).                                      |
+----------------------------------+----------------------------------------------------------+
| Récupérer mot de passe           | En cas d'oubli, l'administrateur demande un lien de       |
|                                  | réinitialisation par e-mail (SMTP) et définit un nouveau |
|                                  | mot de passe sur une page sécurisée.                     |
+----------------------------------+----------------------------------------------------------+
| Gérer profil                     | L'administrateur peut mettre à jour ses informations     |
|                                  | personnelles et ses préférences depuis l'espace Profil.  |
+----------------------------------+----------------------------------------------------------+
| Consulter vue globale            | Permet d'afficher le cockpit administrateur : KPIs       |
|                                  | réseau (expéditions actives, valeur totale, utilisateurs,|
|                                  | file de revue documents), carte mondiale des flux,       |
|                                  | graphique de débit et fil d'activité récent.             |
+----------------------------------+----------------------------------------------------------+
| Consulter analytics globale      | Permet d'accéder au module analytics dédié              |
|                                  | (/dashboard/admin/analytics) : statistiques agrégées sur |
|                                  | les expéditions, documents, délais et performances du    |
|                                  | réseau GlobalTradeX.                                     |
+----------------------------------+----------------------------------------------------------+
| Consulter toutes les expéditions | Permet de lister et ouvrir n'importe quel dossier GTX    |
|                                  | du système, indépendamment du rôle propriétaire, pour    |
|                                  | support, audit ou démonstration.                         |
+----------------------------------+----------------------------------------------------------+
| Suivi GPS en direct              | Permet de visualiser sur une carte l'ensemble des        |
|                                  | expéditions en mouvement suivies par la plateforme.      |
+----------------------------------+----------------------------------------------------------+
| Gérer utilisateurs               | Permet de consulter la liste des comptes, d'en créer    |
|                                  | de nouveaux (e-mail, mot de passe, rôle, profil), de     |
|                                  | modifier le rôle ou l'état actif/inactif, et de supprimer|
|                                  | un compte (sauf le sien). Les comptes créés par l'admin  |
|                                  | sont marqués e-mail vérifié par défaut.                  |
+----------------------------------+----------------------------------------------------------+
| Empêcher auto-désactivation      | L'API interdit à l'administrateur de se désactiver lui-  |
|                                  | même, de changer son propre rôle ou de supprimer son     |
|                                  | propre compte, afin de préserver un accès de secours.     |
+----------------------------------+----------------------------------------------------------+
| Envoyer notifications système  | Permet de diffuser une notification (titre, message,     |
|                                  | type) à des utilisateurs ciblés ou à l'ensemble de la    |
|                                  | base via l'endpoint admin /notify.                       |
+----------------------------------+----------------------------------------------------------+
| Configurer assistant / avatar    | Permet de paramétrer l'assistant vidéo TradeFlow :       |
|                                  | identifiants HeyGen (avatar, voix), message d'accueil,   |
|                                  | activation du service et test de génération depuis       |
|                                  | (/dashboard/admin/assistant-avatar).                     |
+----------------------------------+----------------------------------------------------------+
| Accès transversal aux cockpits   | L'administrateur peut naviguer vers les espaces          |
|                                  | importateur, exportateur, transitaire et courtier        |
|                                  | (RoleGuard) à des fins de support, démonstration ou      |
|                                  | formation, sans changer de compte.                       |
+----------------------------------+----------------------------------------------------------+
| Créer expédition / produits      | Par héritage des droits API, l'administrateur peut       |
|                                  | également créer des expéditions, gérer des produits,     |
|                                  | mettre à jour des statuts et valider des documents       |
|                                  | comme chaque rôle métier le ferait.                      |
+----------------------------------+----------------------------------------------------------+
| Utiliser calculateur logistique  | Permet d'accéder au module calculateur (droits de douane,|
|                                  | estimation fret) pour assistance aux utilisateurs.       |
+----------------------------------+----------------------------------------------------------+
| Consulter documents              | Permet de visualiser et télécharger l'ensemble des       |
|                                  | pièces du réseau, y compris la file de revue courtier.   |
+----------------------------------+----------------------------------------------------------+
| Consulter liste notifications    | Permet de consulter ses propres alertes et l'activité    |
|                                  | récente du système affichée sur le tableau de bord.      |
+----------------------------------+----------------------------------------------------------+
| Gérer préférences notifications  | Permet de configurer les canaux et types d'alertes       |
|                                  | depuis les paramètres du compte.                         |
+----------------------------------+----------------------------------------------------------+
| Messagerie collaborative         | Permet d'échanger avec tous les acteurs via la           |
|                                  | messagerie intégrée pour le support opérationnel.        |
+----------------------------------+----------------------------------------------------------+
| Procurer aide (assistant IA)     | Accès à l'assistant conversationnel TradeFlow et aux     |
|                                  | métriques d'usage de l'assistant côté administration.    |
+----------------------------------+----------------------------------------------------------+
| Se déconnecter                   | Permet de clôturer la session en supprimant le jeton     |
|                                  | d'authentification depuis le profil ou la barre latérale.|
+----------------------------------+----------------------------------------------------------+

Remarques :
- L'administrateur dispose des privilèges cumulés de tous les rôles métier via l'API et
  l'interface ; en production, ces droits doivent être réservés à un petit nombre de comptes.
- Aucun module de facturation / abonnement n'est exposé (hors périmètre du projet).
- Scénario de démonstration : admin@scenario.globaltradex.com — supervision du réseau
  et des cinq comptes scénario (Klaus, Fatima, Karim, Amira).
