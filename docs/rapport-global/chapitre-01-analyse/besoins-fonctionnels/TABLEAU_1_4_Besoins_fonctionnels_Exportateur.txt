================================================================================
TABLEAU 1.4 – Besoins fonctionnels de l'acteur « Exportateur »
Projet : GlobalTradeX — Plateforme de pilotage du commerce international
================================================================================

+----------------------------------+----------------------------------------------------------+
| Fonctionnalité                   | Description                                              |
+----------------------------------+----------------------------------------------------------+
| S'authentifier                   | Pour accéder à son espace, l'exportateur s'authentifie   |
|                                  | avec son adresse e-mail et son mot de passe. L'accès     |
|                                  | n'est accordé qu'après vérification de l'adresse e-mail. |
|                                  | Un jeton JWT sécurisé ouvre le cockpit exportateur       |
|                                  | (/dashboard/exportateur).                                |
+----------------------------------+----------------------------------------------------------+
| Récupérer mot de passe           | En cas d'oubli, l'exportateur demande un lien de         |
|                                  | réinitialisation par e-mail (SMTP). Il définit alors un  |
|                                  | nouveau mot de passe via une page dédiée sécurisée.      |
+----------------------------------+----------------------------------------------------------+
| Gérer profil                     | L'exportateur peut consulter et mettre à jour ses        |
|                                  | informations personnelles (nom complet, téléphone,       |
|                                  | avatar) depuis l'espace Profil.                        |
+----------------------------------+----------------------------------------------------------+
| Consulter cockpit exportateur    | Permet d'afficher un tableau de bord métier : indicateurs|
|                                  | (revenus estimés, expéditions actives, clients importateurs|
|                                  | principaux), graphique d'évolution et recommandations    |
|                                  | commerciales dérivées des dossiers en cours.             |
+----------------------------------+----------------------------------------------------------+
| Gérer catalogue produits         | Permet de consulter, ajouter, modifier et supprimer les  |
|                                  | articles du catalogue export : désignation, code SH,     |
|                                  | prix unitaire, quantité disponible, unité et pays        |
|                                  | d'origine. Une recherche par nom ou code facilite la     |
|                                  | maintenance du référentiel (/dashboard/products).        |
+----------------------------------+----------------------------------------------------------+
| Créer une expédition sortante    | Permet de lancer l'assistant multi-étapes (itinéraire,   |
|                                  | fret, suggestion IA d'itinéraire) avec en plus l'étape   |
|                                  | « Produits » : sélection des lignes du catalogue et      |
|                                  | quantités expédiées. L'exportateur est enregistré comme  |
|                                  | partie exportatrice (exporter_user_id) du dossier.       |
+----------------------------------+----------------------------------------------------------+
| Suivre ses expéditions           | Permet de consulter les expéditions auxquelles il est      |
|                                  | rattaché (créées par lui ou commandées par un importateur |
|                                  | partenaire) : référence GTX, statut logistique, route et  |
|                                  | progression vers la destination.                         |
+----------------------------------+----------------------------------------------------------+
| Consulter détail d'expédition    | Permet d'ouvrir la fiche d'un dossier lié : marchandises  |
|                                  | embarquées, importateur associé, dates, transport et     |
|                                  | historique des événements.                               |
+----------------------------------+----------------------------------------------------------+
| Optimiser itinéraire             | Lors de la création d'expédition, permet de solliciter   |
|                                  | l'assistant IA pour obtenir des propositions de routes     |
|                                  | (modes air, mer, route, rail) avec estimation de fret    |
|                                  | et niveau de risque.                                     |
+----------------------------------+----------------------------------------------------------+
| Téléverser documents commerciaux | Permet de déposer sur un dossier d'expédition les pièces  |
|                                  | d'exportation : facture commerciale, liste de colisage,    |
|                                  | certificat d'origine, etc. (PDF, JPG, PNG — taille max.   |
|                                  | 10 Mo). Les fichiers sont stockés et rattachés au GTX.   |
+----------------------------------+----------------------------------------------------------+
| Consulter documents              | Permet de visualiser l'ensemble des pièces du dossier     |
|                                  | (les siennes et celles des autres acteurs autorisés),      |
|                                  | ainsi que le statut de vérification IA et la décision du   |
|                                  | courtier (vérifié / rejeté / en attente).                |
+----------------------------------+----------------------------------------------------------+
| Supprimer ses documents          | Permet de retirer un document qu'il a lui-même téléversé |
|                                  | tant qu'il en est l'auteur, sous réserve des règles       |
|                                  | d'accès de la plateforme.                                |
+----------------------------------+----------------------------------------------------------+
| Suivi GPS en direct              | Permet de suivre sur une carte les expéditions en        |
|                                  | mouvement liées à ses dossiers exportateurs.             |
+----------------------------------+----------------------------------------------------------+
| Consulter liste notifications    | Permet de consulter les alertes : évolution de statut,   |
|                                  | retour du courtier sur les documents, messages des       |
|                                  | partenaires et anomalies signalées par l'IA.             |
+----------------------------------+----------------------------------------------------------+
| Gérer préférences notifications  | Permet de configurer les canaux et types d'alertes       |
|                                  | (e-mail, SMS si activé) depuis les paramètres du compte. |
+----------------------------------+----------------------------------------------------------+
| Messagerie collaborative         | Permet d'échanger avec l'importateur, le transitaire ou  |
|                                  | le courtier sur un fil de discussion lié au dossier,     |
|                                  | pour coordonner la conformité et la livraison.           |
+----------------------------------+----------------------------------------------------------+
| Procurer aide (assistant IA)     | Un assistant conversationnel (TradeFlow) interagit avec  |
|                                  | l'exportateur pour l'aider sur les codes SH, les         |
|                                  | documents requis à l'export et le suivi de ses dossiers. |
+----------------------------------+----------------------------------------------------------+
| Se déconnecter                   | Permet de clôturer la session en supprimant le jeton     |
|                                  | d'authentification (cookie) depuis le profil ou la barre |
|                                  | latérale.                                                |
+----------------------------------+----------------------------------------------------------+

Remarques :
- L'exportateur ne peut pas faire évoluer le statut logistique d'une expédition (réservé au
  transitaire) ni valider officiellement les documents au nom du courtier en douane.
- L'étape « Produits » du wizard de création lui est exclusive ; l'importateur crée la
  commande sans rattacher le catalogue, l'exportateur complète ensuite le référentiel marchand.
- Scénario de démonstration : Fatima Ben Ali (TunisOlive) — produit « Huile d'olive extra
  vierge » (SH 1509.10), dossier GTX-20240315-00042 vers Klaus Weber (FreshMart).
