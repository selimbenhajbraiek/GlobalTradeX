================================================================================
TABLEAU 1.3 – Besoins fonctionnels de l'acteur « Importateur »
Projet : GlobalTradeX — Plateforme de pilotage du commerce international
================================================================================

+----------------------------------+----------------------------------------------------------+
| Fonctionnalité                   | Description                                              |
+----------------------------------+----------------------------------------------------------+
| S'authentifier                   | Pour accéder à son espace, l'importateur s'authentifie   |
|                                  | avec son adresse e-mail et son mot de passe. L'accès     |
|                                  | n'est accordé qu'après vérification de l'adresse e-mail. |
|                                  | Après authentification, il est redirigé vers son         |
|                                  | tableau de bord importateur sécurisé.                    |
+----------------------------------+----------------------------------------------------------+
| Récupérer mot de passe           | En cas d'oubli, l'importateur demande un lien de         |
|                                  | réinitialisation reçu par e-mail et définit un nouveau   |
|                                  | mot de passe sur une page sécurisée.                     |
+----------------------------------+----------------------------------------------------------+
| Gérer profil                     | L'importateur peut consulter et mettre à jour ses        |
|                                  | informations personnelles (nom complet, téléphone,       |
|                                  | avatar) depuis l'espace Profil.                        |
+----------------------------------+----------------------------------------------------------+
| Consulter cockpit importateur    | Cockpit **Data-Driven** : KPIs métiers sur le périmètre   |
| (KPIs)                           | importateur — (1) **taux de retard à l'arrivée** sur ses |
|                                  | expéditions, (2) **temps moyen de dédouanement**, (3) **taux de conformité documentaire**   |
|                                  | des pièces reçues, (4) **coût estimé vs coût réel**      |
|                                  | (calculateur / dossier). Vue opérationnelle : commandes  |
|                                  | actives, exceptions, liste GTX et fiche détail (ex.      |
|                                  | GTX-20240315-00042).                                     |
+----------------------------------+----------------------------------------------------------+
| Créer une expédition             | Permet de lancer un assistant multi-étapes : saisie de   |
|                                  | l'itinéraire (origine, destination, date, mode de          |
|                                  | transport), caractéristiques du fret (poids, volume,      |
|                                  | valeur, type de marchandise), suggestion d'itinéraires   |
|                                  | par l'assistant TradeFlow et validation finale.          |
|                                  | L'importateur est enregistré comme commanditaire du      |
|                                  | dossier.                                                 |
+----------------------------------+----------------------------------------------------------+
| Suivre ses expéditions           | Permet de consulter, filtrer et rechercher les           |
|                                  | expéditions dont il est commanditaire : référence GTX,   |
|                                  | statut (en attente, en transit, blocage douanier, livré),|
|                                  | parties prenantes (exportateur, transitaire) et          |
|                                  | historique des changements de statut.                    |
+----------------------------------+----------------------------------------------------------+
| Consulter détail d'expédition    | Permet d'ouvrir la fiche complète d'une expédition :     |
|                                  | route, dates, navire / voyage, documents associés,       |
|                                  | notifications liées et indicateurs de progression.       |
+----------------------------------+----------------------------------------------------------+
| Optimiser itinéraire             | Lors de la création d'expédition, permet de solliciter   |
|                                  | l'assistant IA pour obtenir des propositions de routes     |
|                                  | (modes air, mer, route, rail) avec estimation de fret    |
|                                  | et niveau de risque associé.                             |
+----------------------------------+----------------------------------------------------------+
| Utiliser calculateur logistique  | Permet d'estimer les droits de douane (code SH, pays     |
|                                  | d'origine / destination, valeur déclarée) et le coût de  |
|                                  | fret (poids, dimensions) via le **module calculateur**,  |
|                                  | en appui à la décision d'achat.                          |
+----------------------------------+----------------------------------------------------------+
| Consulter documents              | Permet de visualiser les pièces jointes au dossier       |
|                                  | d'expédition déposées par l'exportateur ou le transitaire|
|                                  | (facture commerciale, packing list, certificat d'origine,|
|                                  | connaissement, etc.) sans pouvoir valider au nom du      |
|                                  | courtier en douane.                                      |
+----------------------------------+----------------------------------------------------------+
| Suivi GPS en direct              | Permet de suivre sur une carte les expéditions en        |
|                                  | mouvement (positions simulées / démo) rattachées à ses   |
|                                  | dossiers ou à l'ensemble du réseau selon les droits.     |
+----------------------------------+----------------------------------------------------------+
| Consulter liste notifications    | Permet de consulter les alertes liées à ses expéditions :|
|                                  | changement de statut, blocage douanier, messages des     |
|                                  | partenaires et rappels système.                          |
+----------------------------------+----------------------------------------------------------+
| Gérer préférences notifications  | Permet de configurer les canaux et types d'alertes       |
|                                  | (e-mail, SMS si activé) depuis les paramètres du compte. |
+----------------------------------+----------------------------------------------------------+
| Messagerie collaborative         | Permet d'échanger avec l'exportateur, le transitaire ou  |
|                                  | le courtier sur un fil de discussion lié au dossier,     |
|                                  | afin de coordonner les opérations et lever les blocages. |
+----------------------------------+----------------------------------------------------------+
| Procurer aide (assistant IA)     | Un assistant conversationnel (TradeFlow) interagit avec  |
|                                  | l'importateur pour répondre aux questions sur les        |
|                                  | expéditions, les documents requis et les procédures      |
|                                  | d'importation.                                           |
+----------------------------------+----------------------------------------------------------+
| Se déconnecter                   | Permet de clôturer la session utilisateur de manière     |
|                                  | sécurisée depuis le profil ou le menu de navigation.    |
+----------------------------------+----------------------------------------------------------+

Remarques :
- L'importateur ne peut pas modifier le statut logistique d'une expédition (réservé au
  transitaire) ni valider les documents douaniers (réservé au courtier).
- L'étape « Produits » du wizard de création est réservée à l'exportateur ; l'importateur
  crée la commande et associe ultérieurement l'exportateur via le dossier partagé.
- Référence de démonstration : expédition GTX-20240315-00042 (TunisOlive → FreshMart).
